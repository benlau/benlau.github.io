Logitech Keys-to-go 2

Created Nov 22, 2024 - Last updated: Mar 6, 2025

Seeding 🌱
gadget digital-nomad keyboard

這款鍵盤改變了我對小佈局鍵盤不好用的認知，整體尺寸比 K380 小了一截，但並沒有帶來任何不便，雖然按鍵手感確實比 K380 差，但仍在可接受的範圍內（極限值），我很難想像將來還能有更小而又實用的鍵盤出現。

K380跟Keys-to-go 2的大少比較

整體重要只有222克，因為附有額外的保護外殼，解決了以往這類鍵盤的常見問題——隨身放在背包裡，容易被包內的雜物弄脫鍵帽。

這大少甚至能塞進牛仔褲的褲袋中。（註：不會坐得舒服）

缺點是使用CR2032電池及需要用小型螺絲起子，雖然號稱電池能用36個月，但我已經換了一次電池（後話）

而且，這個外殼還附有磁鐵，增加了一些意想不到的實用功能：

坐在戶外的地上，這樣用鍵盤，還算可以，雖然iPad的角度不太好，但手及頸不會難受。 註：在iPad及Keyboard後面有一本磁吸的 MagEasy A5 Folio 2.0 筆記簿，固定了鍵盤及iPad

這個鍵盤是我進行單車游牧時不可不帶的夥伙

各種問題

但我在使用後遇上了一個問題，在特定情況下，按鍵會出現延遲重覆等現象，有不少用戶投訴過：

Problems with Keys-To-Go 2 : r/logitech

後來甚至無法連線，要拆電池重裝去解決，但拆電池要找來少的螺絲起子，結果我要回公司借工具，為了安全起見，最終還是換了電池。

Logitech的東西越是貴的就越容易出問題，像是 Logitech MX Master 3 Mouse就令人很失望。

至令為止，我已經進行過三次電池拆卸重裝的動作。

熱鍵

  • Press Fn + O for three seconds to trigger iPadOS / iOS / macOS layout
  • Press Fn + G for three seconds to trigger Android layout
  • Press Fn + P for three seconds to trigger Windows layout
  • Press Fn + C for three seconds to trigger ChromeOS layout
  • Fn + Esc to switch from default key functions to standard F1 - F12 function keys.

Other Shrotcuts

  1. Fn Lock: Esc - Emoji
  2. Fn +F1 = Easy-Switch 1
  3. Fn + F2 = Easy-Switch 2
  4. Fn + F3 = Easy-Switch 3
  5. Fn + F6 = Mission Control
