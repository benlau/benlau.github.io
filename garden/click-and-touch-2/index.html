click&touch2開箱 — 在鍵帽上加入觸控功能是天才還是蠢材的設計？
click&touch2是一款把觸控板做進鍵帽下的小型鍵盤，外表就跟一般小型鍵盤一樣，但只要讓手指在鍵帽表面滑動，便能驅動鼠標，單擊、右擊、雙擊、滾軸等操也能無障礙地進行，而不需要有一塊額外的面板作觸控之用。

優點

  • 測試一下，打字方面，手感跟Logitech k380差不多。
  • 足以作為日常使用的鍵盤

缺點

  • 低速的Scrolling很困難
  • 電量消耗太快
  • 曾經發生過Fn + Ctrl交換的情況，最終要靠factry reset解決

Shortcut Keys

Hold F1~F3: Bluetooth pairing F1, F2,F3 switch channel Fn+Tab: Turn off touch function Fn+Esc: Switch Function key usage Fn+Delete: Reset to factory setting F6: setting mode, press again to disable the setting mode Fn+W MacOS preset
