Only the Paranoid Survive – 唯偏執狂得以倖存

「這是一本精彩的書，也是一本危險的書，令人深思。」──彼得‧杜拉克（Peter Drucker），管理大師

這是我在中學時候讀過的一本書，當時是第一版，記得在是校內的書展上，翻起了這本書，被封面的一句話「這是一本精彩的書，也是一本危險的書，令人深思」所吸引，結果就買了下來。

書的內容主要講述當年Intel轉型的經驗。

當時Intel的名字能跟記憶體畫上等號，但在得到日本政府支持的半導體公司所發起的減價戰下打到不要不要，與此同時Intel制作了4004處理器後又開發了8086/8088等處理器，擺在Andrew S. Grove眼前出現了兩個選擇，死守記憶體或投入未知的處理器市場。

  • 記億有點模糊，不太記得那時8088是否已面世，求指正

最終Intel決定轉型，獲得了空前的成功，選錯了的話就會被埋沒在歷史之中。

內容是相當之精彩的，但想到現在的Intel的處景就更令人感到唏噓，在ARM剛崛起之時，曾有個王者般的存在，就是Intel PXA系列，無論性能、軟件支援都是最棒的（價錢也⋯⋯）

後來Intel卻放棄ARM死守X86，搞什麼x86 for mobiel，結果完全失敗，慢性地走向下坡，這間公司無復當年的精神。
